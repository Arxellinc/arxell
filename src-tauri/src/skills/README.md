# Portable Agent Skills Pack (2026)

This folder mirrors the Claude Code skills in `.claude/skills/`, but in a **runtime-agnostic** format you can copy into other agent systems (Claude Code, “agent runners”, internal frameworks, etc.).

## How to use elsewhere
- Treat each file as a **capability playbook**: triggers, workflow, output templates, and stop conditions.
- Adapt the metadata (frontmatter / manifests) to your runtime.
- Keep the **core body** identical so behavior stays consistent across environments.
