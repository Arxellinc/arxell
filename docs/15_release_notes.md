# Release Notes

## Current Release
- Version: `0.8.0`
- Release date: Not explicitly encoded in source files reviewed.

## 0.8.0 Highlights (from source snapshot)
### Added/Expanded
- Multi-panel workspace tool architecture (tasks, MCP, system, terminal, help, avatar, etc.).
- API model config management with verification and primary selection.
- Local model management and runtime engine installer flow.
- Voice pipeline with VAD + STT/TTS engine checks and diagnostics.
- Task and MCP JSON views for agent-readable context.

### Operational/UX
- Sidebar system resource display and diagnostics integration.
- Log event forwarding and in-app terminal log tabs.
- Browser panel proxy/reader/markdown fetch modes.

### Known constraints
- Theming system is incomplete and largely hardcoded.
- Some safety boundaries are UI-mediated and should be hardened server-side for stricter security.
